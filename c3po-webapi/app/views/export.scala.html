Export

Currently, there are three possiblities for data export. First, you can export a machine readable profile of your current filter, which can be used as input in the Planning Tool Plato.

Second, you can export a '.csv' file containing all the data for the selected collection, or third - a '.csv' file containing only the current filter. In other words a sparse matrix over all known properties for each element. This '.csv' file is very helpful for further analysis with the help of spreadsheet software such as Libre Office or Microsoft Excel.
